Personal Finance Tracker is a lightweight, self-hosted web application for recording daily expenses, organizing them by category, setting monthly budgets, and visualizing spending patterns over time. It runs entirely on your own machine — no accounts, no subscriptions, no data sent to third-party services.

## Technology stack

| Layer | Technology | Purpose |
|---|---|---|
| Backend | [Flask](https://flask.palletsprojects.com/) | HTTP server and routing |
| Database | SQLite | Local, file-based data storage |
| Charts | [Chart.js](https://www.chartjs.org/) | Interactive spending visualizations |
| Frontend | Vanilla JavaScript | UI interactions without a framework |

Flask serves a small set of JSON API endpoints (`/api/expenses`, `/api/budgets`, `/api/stats`) alongside a single-page HTML frontend. SQLite stores all data in a single `database.db` file, so backups are as simple as copying one file.

## Who it is for

Personal Finance Tracker is designed for individuals who want to:

- Track spending without sharing financial data with a cloud service
- Run a simple finance tool on a home server or personal laptop
- Customize or extend the codebase to fit their own workflow
- Learn how a Flask + SQLite application is structured

It is intentionally single-user and self-hosted. There is no authentication layer, so it should be run on a trusted local network or behind appropriate access controls if exposed beyond localhost.
