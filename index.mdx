## What is Personal Finance Tracker?

Personal Finance Tracker is a lightweight, self-hosted web application that helps you take control of your finances. Built with Flask and SQLite, it gives you a clean dashboard to record every expense, enforce budget discipline by category, and understand your spending patterns through interactive charts.

There are no subscriptions, no external accounts, and no data leaving your machine. Run it locally and own your financial data completely.

## Key capabilities

<CardGroup cols={3}>
  <Card title="Expense logging" icon="plus">
    Record transactions with amount, category, date, and an optional description. Edit or remove any entry at any time.
  </Card>
  <Card title="Category budgets" icon="chart-bar">
    Assign a monthly spending limit to any of the seven built-in categories and watch real-time progress bars fill up.
  </Card>
  <Card title="Spending insights" icon="chart-pie">
    Interactive doughnut and bar charts show where your money goes and how close you are to each budget limit.
  </Card>
  <Card title="Dashboard stats" icon="gauge">
    See your total spend, transaction count, and per-expense average at a glance on the main dashboard.
  </Card>
  <Card title="REST API" icon="webhook">

  <Card title="Local SQLite storage" icon="database">

## How it works

<Steps>
  <Step title="Install dependencies">
    Install Python and the Flask package, then clone the repository.
  </Step>
  <Step title="Start the server">
    Run `python app.py` inside the `expense-tracker/` directory to start the local server on port 5000.
  </Step>
  <Step title="Open the dashboard">
    Navigate to `http://localhost:5000` in your browser to access the full UI.
  </Step>
  <Step title="Add your first expense">
    Fill in the amount, select a category, pick a date, and click **Add Expense**.
  </Step>
  <Step title="Set budget limits">
    Use the **Set Budget Limits** form to define monthly caps for each spending category.
  </Step>
</Steps>

<Info>
  Personal Finance Tracker stores all data locally in `expense-tracker/database.db`. Back up this file regularly to preserve your records.
</Info>
