This guide walks you through installing, starting, and making your first entries in Personal Finance Tracker.

## Prerequisites

- Python 3.8 or later
- `pip` (included with Python 3.4+)
- A terminal and a web browser

<Note>
  Verify your Python version before starting: `python --version` or `python3 --version`.
</Note>

## Setup

<Steps>
  <Step title="Get the source code">
    Download or clone the project to your machine.

    <CodeGroup>

    ```bash git clone
    git clone https://github.com/your-org/personal-finance-tracker.git
    ```

    ```bash download zip
    # Download the ZIP from the repository and unzip it
    unzip personal-finance-tracker.zip
    ```

    </CodeGroup>
  </Step>

  <Step title="Navigate to the app directory">
    The Flask application lives inside the `expense-tracker/` subdirectory.

    ```bash
    cd personal-finance-tracker/expense-tracker
    ```
  </Step>

  <Step title="Install Flask">
    Install the only required dependency.

    ```bash
    pip install flask
    ```

    <Tip>
      Use a virtual environment to keep your system Python clean:

      ```bash
      python -m venv .venv
      source .venv/bin/activate   # macOS/Linux
      .venv\Scripts\activate      # Windows
      pip install flask
      ```
    </Tip>
  </Step>

  <Step title="Start the development server">
    Run the application with Python.

    ```bash
    python app.py
    ```

    You should see output similar to:

    ```
    * Running on http://127.0.0.1:5000
    * Debug mode: on
    ```

    The SQLite database file (`database.db`) is created automatically on first run.
  </Step>

  <Step title="Open the app in your browser">
    Navigate to `http://localhost:5000` in any web browser. The main dashboard loads with an empty expense list and chart area.

    <Note>
      Leave the terminal running while you use the app. Press `Ctrl+C` to stop the server when you are done.
    </Note>
  </Step>

  <Step title="Add your first expense">
    1. Click **Add Expense** (or the equivalent button in the UI).
    2. Enter an **amount** — for example, `12.50`.
    3. Select a **category** from the dropdown: Food, Transport, Shopping, Entertainment, Bills, Health, or Other.
    4. Choose a **date** (defaults to today).
    5. Optionally enter a **description** such as `Lunch at the market`.
    6. Click **Save**.

    The new entry appears in the expense list and the chart updates to reflect your spending.
  </Step>

  <Step title="Set your first budget limit">
    1. Go to the **Budgets** section of the app.
    2. Select the category you want to cap — for example, **Food**.
    3. Enter a monthly limit — for example, `300`.
    4. Click **Save Budget**.

    The app will now show how much of your Food budget you have used whenever you view that category's spending.

    <Tip>
      Start with one or two categories where overspending is most common. You can add or adjust limits at any time.
    </Tip>
  </Step>
</Steps>

## What's next

Once you have a few expenses recorded, explore the stats view to see a breakdown of spending by category. The Chart.js visualizations update automatically as you add data.
